# Relações de Ordem

----------------------------------------------------------------

Definição:
----------

Uma relação binária em um conjunto A que seja reflexiva, antisimétrica e transitiva 
é dita ser uma relação de ordem parcial (ordenação parcial) em A.

----------------------------------------------------------------

Exemplos:
----------

R1 = {(x, y)| x ≤ y} (em N)

R2 = {(A, B)| A ⊆ B} (em P(N))

R3 = {(x, y)| x divide y} (em Z+)
